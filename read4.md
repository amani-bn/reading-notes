# HTML5 is introducing a new set of
elements that help define the structure of
a page.
## HTML5 introduces a new set of elements that allow you to divide up the
parts of a page.

### these elements are :
- The <header> and <footer>
- The <nav> element
- The <article> element
- The <aside> element
- The <section> element
- the <hgroup> element
- the <div> element
